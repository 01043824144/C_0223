C언어 심화과정 BookCAfe Project 만들기

=============


1. bookcafeproject 프로젝트 생성
2. IO(콘솔) 책추가, 책삭제, 책목록출력, 책목록저장, 책찾기, 책목록 불러오기
		책추가-책삭제-책목록출력-찾기-> 콘솔
		책목록저장-목록불러오기 -> FileIO
3. 1차로는 배열 2차로는 구조체 3차로는 구조체+동적메모리 할당

------------------------------------------------------------

>1
>책이름을 입력해 주세요. : 정의란 무엇인가.
>저자를 입력해 주세요 : 마이클 샌댈
>책이 저장되어 있습니다.

------------------------------------------------------------

>add 정의란무엇인가 마이클샌댈
>책이 저장되었습니다.

> add, list, del, find  (추가, 목록출력, 삭제, 찾기)
> h (도움말 나오게 하는 명령어)
> q (프로그램 종료)
> save [파일명] - 파일로 데이터 저장
> read [파일명] - 파일 읽기
